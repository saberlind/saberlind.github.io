    Java 内置类

    Awaken 11/23/2019 JavaSE

    # System类

    java.lang.System类中提供了大量的静态方法和静态变量，可以获取与系统相关的信息或系统级操作。System里最常使用的是out属性，用来实现输出，例如System.out.println。

    System里常见的方法和属性还有:

    • out属性：通常用来实现输出，例如System.out.print()
    • in属性：通常用来接收用户的输入，例如Scanner scanner = new Scanner(System.in);
    • public static void setOut(PrintStream out)：用来修改标准输出位置。
    • public static void setIn(InputStream in)：用来修改标准输入位置。
    • public static void gc()：通知JVM，希望进行一次垃圾回收，JVM收到消息后，会在适当的实际进行垃圾回收。
    • public static void exit(int status)：以指定的退出码结束当前系统。
    • public static long currentTimeMillis()：返回1970-01-01 00:00:00 距离现在的毫秒数。

    # 日期和时间

    Java里表示以及操作日期和时间的主要有这么几个类：

    • java.util.Date类，用来表示特定的瞬间，精确到毫秒。
    • java.text.DateFormat类，用来实现字符串（String）和日期（Date）之间相互转换。
    • java.util.Calendar类，替换了很多Date类里的功能，这个类将能用到的时间信息都封装成为静态成员，方便获取。

    除了上述的类以外，JDK8的java.time包还提供了其他的类来操作日期时间:

    • LocalDate类，用来表示日期。
    • LocalTime类，用来表示时间。
    • LocalDateTime类，用来日期和时间。
    • DateTimeFormatter类，用来实现日期时间和字符串之间的相互转换。
    • Instant类，用来表示时间戳（很少使用）

    # Date类

    Date类用来表示日期，它里面有很多的方法都被Calendar类取代了。

    # 构造方法

    • public Date()：创建一个当前时间的日期对象。
    • public Date(long date)：分配Date对象并初始化此对象，以表示自从标准基准时间（称为“历元（epoch）”，即1970年1月1日00:00:00 GMT）以来的指定毫秒数。

    # 常用方法

    • public long getTime() ：把日期对象转换成对应的时间毫秒值。
    • public boolean after(Date when)：判断是否在when时间之后。
    • public boolean before(Date when)：判断是否在when时间之前。
    • public int compareTo(Date anotherDate)：判断时间和指定时间的大小关系。

    # DateFormat类

    java.text.DateFormat 是Format类的抽象子类，我们通过这个类可以帮我们完成日期和文本之间的转换,也就是可以在Date对象与String对象之间进行来回转换。

    • 格式化：按照指定的格式，从Date对象转换为String对象。
    • 解析：按照指定的格式，从String对象转换为Date对象。

    # 构造方法

    DateFormat是一个抽象类，不能直接创建对象，通常我们会选择创建一个SimpleDateFormat类型的对象。

    public SimpleDateFormat(String pattern)：通常会传入一个格式化字符串参数。

    SimpleDateFormat sdf = new SimpleDateFormat("yyyy-mm-dd HH:MM:SS");
    1

    参数pattern是一个字符串，代表日期时间的自定义格式。

    # 常见方法

    • public String format(Date date)：将Date对象格式化为字符串。

    • public Date parse(String source)：将字符串解析为Date对象。

      // 创建一个 SimpleDateFormat 对象
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-mm-dd HH:MM:SS");
// 创建一个 Date 对象
Date date = new Date();

// 将当前时间以指定的格式转换成为字符串
String str1 = sdf.format(date);
System.out.println(str1);  // 2020-03-15 11:22:35

// 按照格式创建一个字符串对象
String dateStr = "2008-08-08 12:34:45";

// 将字符串加载成为一个 Date 对象
Date newDate = sdf.parse(dateStr);

    # Calendar类

    java.util.Calendar是日历类，在Date后出现，替换掉了许多Date的方法。该类将所有可能用到的时间信息封装为静态成员变量，方便获取。日历类就是方便获取各个时间属性的。

    # 构造方法

    Calendar为抽象类，由于语言敏感性，Calendar类在创建对象时并非直接创建，而是通过静态方法创建，返回子类对象。

    • public static Calendar getInstance()：使用默认时区和语言环境获得一个日历。

    # 常见方法

    Calendar类里通常使用两个方法:

    • public int get(int field)：获取指定字段（年月日等）的值。
    • public void set(int filed, int vale)：给指定的字段设置值。

    Calendar里的字段:

    Calendar类中提供很多成员常量，代表给定的日历字段：

    字段值 含义
    YEAR 年
    MONTH 月（从0开始，可以+1使用）
    DAY_OF_MONTH 月中的天（几号）
    HOUR 时（12小时制）
    HOUR_OF_DAY 时（24小时制）
    MINUTE 分
    SECOND 秒
    DAY_OF_WEEK 周中的天（周几，周日为1，可以-1使用）

    # JDK8的日期时间类

    JDK8的java.time包里还提供了LocalDate类，LocalTime类,以及LocalDateTime类来分别用来表示日期，时间以及日期和时间。这三个类的构造方法都是私有的，不要直接创建对象，都是通过静态方法now来获取到当前的日期和时间。

    获取到日期时间对象以后，可以使用 get(filed)方式获取到指定的字段，还可以使用 getXXX直接获取到指定数据。

    LocalDate ld = LocalDate.now();
System.out.println(ld.get(ChronoField.DAY_OF_WEEK));
System.out.println(ld.getDayOfYear());
System.out.println(ld.getMonth());

LocalTime lt = LocalTime.now();
System.out.println(lt.get(ChronoField.HOUR_OF_DAY));
System.out.println(lt.getMinute());

LocalDateTime ldt = LocalDateTime.now();
System.out.println(ldt.get(ChronoField.MINUTE_OF_HOUR));
System.out.println(ldt.getDayOfMonth());

    除了上述三个类以外，JDK8还提供了Instant类来表示时间，它是精确到纳秒的。

    // 获取到当前时间
Instant instant = Instant.now();
System.out.println(instant);

// 使用指定的数字创建时间
Instant day = Instant.ofEpochSecond(1000000);
System.out.println(day);

// 进行日期计算
System.out.println(day.plus(Duration.ofDays(5)));

// 将字符串加载成为 Instant 对象
Instant ins = Instant.parse("2021-01-01T12:00:10.123Z");
System.out.println(ins);

    除了使用SimpleDateFormat类对时间进行格式化以外，还可以使用DateTimeFormatter类来进行格式化。调用DateTimeFormatter类的静态方法ofPattern传入一个字符串格式，就可以获取到一个日期时间格式化器。

    DateTimeFormatter dtf = DateTimeFormatter.ofPattern("yyyy年MM月dd日 HH点mm分ss秒");
LocalDateTime ldt = LocalDateTime.parse("2021年01月08日 20点12分34秒", dtf);
System.out.println(ldt.getYear());

    # BigInteger和BigDecimal(处理精度丢失)

    正常情况下一个整数最多只能放在long类型之中，但是我们都知道long类型的长度是有限（-263到263-1之间）。如果一个整数超过了long的范围以后该怎么表示呢？同样，如果一个浮点数超出了double的范围又该如何表示？

    此时就需要BigInteger和BigDecimal用来表示超大整数和浮点数。

    在创建BigInteger和BigDecimal时，数字参数通常使用字符串的形式传入。

    BigInteger x = new BigInteger("1234");
BigDecimal y = new BigDecimal("34.56");
