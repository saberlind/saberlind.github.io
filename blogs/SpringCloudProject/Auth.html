    # 用户认证

    CAS单点登录：服务端保存用户登录状态

    token模式单点登录：jwt(也可以自定义 加密格式) 客户端保持登录状态

    字符串处理出现乱码的原因： 处理后的字节在码表中没有对应的字符 或者对应的是一个特殊字符

    一个字节由8bit组成： [ 0000 0000 ] ~[ 1111 1111 ] 加密后的字节可能第一位是负数，就会出现乱码

    jwt使用密钥对jwt有效载荷+头处理生成签名时，最终的结果没有乱码，因为它生成签名后使用BASE64对结果进行了处理。

    BASE64：不是为了加密，为了让数据可读不会出现乱码

    处理过程：

    ​ base64处理字符串时，会将每三个字节分为一组[ 1010 1111 ] [ 0011 1010 ] [ 0101 0101 ] [ 1100 1110],如果最后的字节不足三个，使用等号补齐

    ​ [ 1010 1111 ] [ 0011 1010 ] [ 0101 0101 ] [ 1100 1110] = =

    ​ 然后将每三个字节划分为4个字节，每个字节只有6位，最后它在每个字节的高位填充两个0

    [ 1010 1111 ] [ 0011 1010 ] [ 0101 0101 ]

    [0010 1011] [0011 0011] [0010 1001] [0001 0101]

    ​ 每个字节的取值范围： 000000~ 111111 64种可能

    base64自己提供了码表： 0~9 , a~z , A~Z 62 、 + 和 / 和一个不会解析的特殊符号=

    base64处理过的字节只能从自己的码表中映射字符，一定是可读的

    base64将字符串还原：每四个字节分为一组，高位的两个0去掉，合并为三个字节

    import com.sun.org.apache.xml.internal.security.utils.Base64;
@Test
public void testBase64() throws Base64DecodingException {
    String str = "尚硅a";
    String encodeStr = Base64.encode(str.getBytes(StandardCharsets.UTF_8));
    System.out.println(encodeStr);
    byte[] bytes = Base64.decode(encodeStr);
    System.out.println(bytes.length);
    String decodeStr = new String(bytes, StandardCharsets.UTF_8);
    System.out.println(decodeStr);
}

    # 用户认证流程

    1、用户在登录页面提交登录请求给ucenter服务

    ucenter微服务处理登录成功，将用户数据封装为jwt响应给浏览器

    2、客户端得到异步请求响应的R中的jwt，缓存jwt（cookie:浏览器每次访问时会自动携带，默认保存在浏览器的运行内存中、localStorage:不会自动携带、持久化保存）

    3、客户端再访问后端服务器需要验证登录状态的接口时，可以将jwt取出存在请求头或者请求参数中提交给后端

    4、后端接收到请求，获取报文中的jwt，进行验证

    • 如果jwt过去、为空、验证签名失败 都认为用户未登录，响应需要登录的R（前端得到需要登录的响应，清空缓存，跳转到登录页面）
    • 如果jwt验证通过，jwt解析得到的id就代表当前客户端的身份

    @Slf4j
public class JwtHelper {

    //过期时间，毫秒，一天
    private static long tokenExpiration = 24*60*60*1000;
    //秘钥
    private static String tokenSignKey = "123456";

    private static Key getKeyInstance(){
        SignatureAlgorithm signatureAlgorithm = SignatureAlgorithm.HS256;
        byte[] bytes = DatatypeConverter.parseBase64Binary(tokenSignKey);
        return new SecretKeySpec(bytes,signatureAlgorithm.getJcaName());
    }

    public static String createToken(JwtInfo jwtInfo){

        String JwtToken = Jwts.builder()
                .setHeaderParam("typ", "JWT")
                .setHeaderParam("alg", "HS256")
                .setSubject("GULI-USER")//主题
                .setExpiration(new Date(System.currentTimeMillis() + tokenExpiration))
                .claim("id", jwtInfo.getId())//用户id
                .claim("nickname", jwtInfo.getNickname())//用户昵称
                .claim("avatar", jwtInfo.getAvatar())//用户头像
                .signWith(SignatureAlgorithm.HS256, getKeyInstance())
                .compact();

        return JwtToken;
    }

    /**
     * 判断token是否存在与有效
     * @param request
     * @return
     */
    public static Claims checkToken(HttpServletRequest request) {
        String token = request.getHeader("token");
        if(StringUtils.isEmpty(token)) {
            throw new GuliException(ResultCodeEnum.LOGIN_AUTH);
        }
        try {
            Jws<Claims> claimsJws = Jwts.parser().setSigningKey(getKeyInstance()).parseClaimsJws(token);
            Claims claims = claimsJws.getBody();
            return claims;
        } catch (Exception e) {
            log.error(ExceptionUtils.getStackTrace(e));
            throw new GuliException(ResultCodeEnum.LOGIN_AUTH);
        }
    }

    /**
     * 判断token是否有效
     * @param token
     * @return
     */
    public static boolean checkToken(String token) {
        if(StringUtils.isEmpty(token)) {
            return false;
        }
        try {
            Jwts.parser().setSigningKey(getKeyInstance()).parseClaimsJws(token);
            return true;
        } catch (Exception e) {
            return false;
        }
    }

    /**
     * 根据token获取会员id
     * @param request
     * @return
     */
    public static String getId(HttpServletRequest request) {
        Claims claims = checkToken(request);
        return (String)claims.get("id");
    }

    /**
     * 根据token获取会员信息
     * @param request
     * @return
     */
    public static JwtInfo getJwtInfo(HttpServletRequest request) {
        Claims claims = checkToken(request);
        JwtInfo jwtInfo = new JwtInfo(claims.get("id").toString(), claims.get("nickname").toString(), claims.get("avatar").toString());
        return jwtInfo;
    }

    public static void main(String[] args) {

        JwtInfo jwtInfo = new JwtInfo();
        jwtInfo.setId("1");
        jwtInfo.setNickname("Helen");
        jwtInfo.setAvatar("1.jpg");

        String token = JwtHelper.createToken(jwtInfo);
        System.out.println(token);
    }
}

    @Override
public String login(String mobile, String password) {
    /*
            以后登录密码校验，一般先使用账号取出数据
             然后再验证密码是否正确

             因为有些加密方式 同一个明文每次加密的结果都不一样[BCryptPasswordEncoder]
         */
    LambdaQueryWrapper<Member> queryWrapper = new LambdaQueryWrapper<>();
    queryWrapper.eq(Member::getMobile , mobile);
    //  queryWrapper.eq(Member::getPassword , password);
    Member member = baseMapper.selectOne(queryWrapper);
    if(member==null){
        throw new GuliException(ResultCodeEnum.LOGIN_MOBILE_ERROR);
    }
    if(!MD5.encrypt(password+ServiceConsts.ENCODE_KEY).equals(member.getPassword())){
        throw new GuliException(ResultCodeEnum.LOGIN_PASSWORD_ERROR);
    }
    //将查询到的用户转为jwt返回
    JwtInfo jwtInfo = new JwtInfo();
    jwtInfo.setId(member.getId());
    jwtInfo.setAvatar(member.getAvatar());
    jwtInfo.setNickname(member.getNickname());
    return  JwtHelper.createToken(jwtInfo);
}

    Header.vue

    <script>
import local from '~/utils/local'
import ucenterApi from '~/api/ucenter'
export default {
  data() {
    return {
      user: undefined
    }
  },
  mounted() {
    // 判断 如果路径中包含 token参数，代表用户是wx登录成功后跳转回当前页面
    if (this.$route.query.token) {
      local.save('GULI-TOKEN', this.$route.query.token)
      window.location = '/'
    }
    // 如果localstorage中缓存了guli-user，不再查询直接使用
    const guliUser = local.get('GULI-USER')
    if (guliUser) {
      this.user = JSON.parse(guliUser) // 需要将json字符串转为json对象
      return
    }
    // 从localstorage中获取GULI-TOKEN，然后发送请求判断GULI-TOKEN是否有效，查询token中用户的信息
    const guliToken = local.get('GULI-TOKEN')
    if (guliToken) {
      ucenterApi.getUserInfo(guliToken).then(r => {
        this.user = r.data.item
        // 查询到用户信息后可以缓存到浏览器中，避免刷新页面再次查询
        local.save('GULI-USER', JSON.stringify(this.user))// json对象不能直接缓存，必须转为json字符串
      })
    }
  },
  methods: {
    logout() {
      local.remove('GULI-TOKEN')
      local.remove('GULI-USER')
      // this.$router.push('/')
      // 清楚缓存后  让整个页面刷新
      window.location = '/'
    }
  }
}
</script>
