    # Gateway网关

    # 1.作用

    1. 统一后端的访问入口：
      • 统一日志处理、统一限流、统一鉴权(用户登录状态)、白名单黑名单
    2. 路由：负载均衡
      • 根据访问网关的请求地址，将请求分发给目标微服务处理
    3. 过滤：
      • 全局过滤器：对所有的路由都会过滤
      • 局部过滤器：哪个路由配置对哪个起作用

    springcloud zuul2:

    ​ eureka+zuul2

    springcloud gateway：基于NIO(netty+webflux：和servlet一样都可以处理web请求)

    # 4.网关过滤器的使用

    # 1.统一跨域配置

    创建GlobalGatewayConfig并配置CorsWebFilter

    @Component
public class GlobalGatewayConfig {

    //将gateway提供的跨域filter对象注入到容器中
    @Bean
    public CorsWebFilter corsWebFilter(){

        //一个路径的跨域配置
        CorsConfiguration config = new CorsConfiguration();
        config.setAllowCredentials(true);//是否允许携带cookie
        config.addAllowedHeader("*");//请求头
        config.addAllowedMethod("*");//所有请求方式
        config.addAllowedOrigin("*");//所有服务器
        
        UrlBasedCorsConfigurationSource source = new UrlBasedCorsConfigurationSource();
        source.registerCorsConfiguration("/**" , config);
        return new CorsWebFilter(corsConfigurationSource);
    }

    # 2.统一鉴权

    为了减少网关的访问压力，后端有些接口用户访问时可以不需要鉴权。

    步骤：

    ​ 2.1判断是否需要鉴权；一般在后端需要鉴权的接口路径中添加一层路径【/auth】

    ​ 如果请求路径中包含/api/**/auth/**则该接口需要鉴权

    ​ 2.2鉴权：

    ​ 使用JwtHelper对请求头中的token进行鉴权

    ​ 2.3鉴权失败，返回需要登录（不用将请求交给目标微服务处理）

    ​ 2.4鉴权通过，放行请求，将请求交给目标微服务处理

    创建GlobalAuthFilter

    @Component
public class AuthGlobalFilter implements GlobalFilter, Ordered {

    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        System.out.println("全局过滤器");
        ServerHttpRequest request = exchange.getRequest();
        String path = request.getURI().getPath();

        //谷粒学院api接口，校验用户必须登录
        AntPathMatcher antPathMatcher = new AntPathMatcher();
        if (antPathMatcher.match("/api/**/auth/**",path)){
            System.out.println("需要鉴权的接口");

            List<String> tokenList = request.getHeaders().get("token");
            //没有token
            if (null==tokenList){
                ServerHttpResponse response = exchange.getResponse();
                return out(response);
            }

            //token校验失败
            boolean isOK = JwtHelper.checkToken(tokenList.get(0));
            if (!isOK){
                ServerHttpResponse response = exchange.getResponse();
                return out(response);
            }
        }

        //放行
        return chain.filter(exchange);
    }
    /**
     * 封装响应体内容，鉴权失败后返回。
     * @param response
     * @return
     */
    private Mono<Void> out(ServerHttpResponse response) {

        JsonObject message = new JsonObject();
        message.addProperty("success", ResultCodeEnum.LOGIN_AUTH.getSuccess());
        message.addProperty("code", ResultCodeEnum.LOGIN_AUTH.getCode());
        message.addProperty("data", "");
        message.addProperty("message", ResultCodeEnum.LOGIN_AUTH.getMessage());
        //将json对象转为json字符串
        String jsonObject = message.toString();
        //将json作为响应体写入到响应报文中
        DataBuffer buffer = response.bufferFactory().wrap(jsonObject.getBytes(StandardCharsets.UTF_8));
        //设置响应头，指定编码，否则在浏览器中会中文乱码
        response.getHeaders().add("Content-Type", "application/json;charset=UTF-8");
        //输出http响应
        return response.writeWith(Mono.just(buffer));
    }
    @Override
    public int getOrder() {
        //定义当前过滤器的优先级，值越小，优先级越高
        return 0;
    }
}
